## Live result

```tsx live
function example() {
  const [date, setDate] = useState(null);
  const [time, setTime] = useState(null);
  const [appt, setAppt] = useState(null);

  return (
    <div style={{ maxWidth: '28rem' }}>
      <Field label="Profile photo">
        <File hasName buttonLabel="Choose a file…" />
      </Field>
      <DateInput label="Date of birth" value={date} onChange={setDate} />
      <TimeInput
        label="Preferred contact time"
        value={time}
        onChange={setTime}
      />
      <DateTimeInput label="Appointment" value={appt} onChange={setAppt} />
    </div>
  );
}
```

:::note
The date/time inputs are controlled with a `Date | null` value and an `onChange(date)` handler,
just like the text inputs — only the widget differs.
:::
